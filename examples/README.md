# Examples

These examples exercise deterministic local flows and run from the repository
root with `PYTHONPATH=src`.

- `client/tool_discovery.py`: list registered MCP tools from the in-process app.
- `su2/session_lifecycle.py`: create/inspect/close a session.
- `su2/results_snapshot.py`: create deterministic result files and inspect them.
- `server/http_launch_config.py`: show parsed HTTP transport launch settings.
